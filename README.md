# husoyayazar

Tek sayfalik Three.js deneyi:
- Arka planda dinamik yildiz alani
- `husospace.png` tabanli, dusuk opaklikli takimyildiz formu
- Star Wars acilis temasi gibi kayan yemek metinleri
- Mouse ile kontrol edilen smooth karadelik (cekme/itme)
- Cihaz gucune gore otomatik kalite ayari (yildiz/adet)

## Kontroller
- Sol tik: ekstra yemek satiri olusturur
- Sag tik basili: karadelik cekmek yerine iter
- Mobil uzun basma: itme modu
- Bosluk: gecici karadelik patlamasi
- Sistem imleci gizlidir; karadelik imlec gibi kullanilir

## Dosya yapisi
- `index.html`: sayfa kabugu
- `main.js`: tum sahne/animasyon mantigi
- `husospace.png`: takimyildiz katmani icin girdi
